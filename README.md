## How to run (local development)
Depending on how the site is built, use one of the following:

Static site (HTML/CSS/JS)
1. Clone the repo:
   git clone https://github.com/rksaykot999/RK-Saykot-Portfolio.git
2. Open `index.html` in your browser.

React or other Node-based site
1. Clone the repo:
   git clone https://github.com/rksaykot999/RK-Saykot-Portfolio.git
2. Install dependencies:
   npm install
3. Start the dev server:
   npm start
4. Build for production:
   npm run build

Adjust the commands above to match the project's actual setup (Yarn, pnpm, or a different build script).
